# Seminar Hall Digital Twin

A 3D recreation of the PES CAVE Labs seminar hall, modeled in Blender and brought to life in Unity with interactive lighting, ceiling fans, and multiple camera views.

## Contents
- `SeminarHall.blend` — Blender source file
- `SeminarHall.fbx` — Exported model
- `Assets/Scripts/` — C# scripts (FanRotate, RoomLights, CameraSwitcher)
- `Scenes/` — Unity scene file
- `Demo.mp4` — 1-2 minute demo video

## Interactions Implemented
1. **Ceiling Fans** — All fans rotate continuously (C# script)
2. **Room Lights Toggle** — 5 independent UI buttons toggle each row of ceiling lights on/off
3. **Camera Views** — UI buttons switch between 3 preset camera angles
